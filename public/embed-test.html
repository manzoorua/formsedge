FormsEdge Embed Test Page

This page demonstrates how to embed FormsEdge forms on external websites.

✓ Testing Instructions:
1. Go to your form's Embed SDK tab in FormsEdge
2. Copy the complete embed code (both the <div> and <script> tags)
3. Replace lines 66-74 below with your actual embed code
4. Open this file in a browser to test
5. Your form should load correctly with all fields visible
⚠️ Critical: Do NOT use old script URLs like https://formsedge.com/formsedge-embed.js. Always use the environment-specific URL from your Embed SDK tab (e.g., https://abc123.lovableproject.com/formsedge-embed.js). Using the wrong script URL will cause "File not found" or "null/f/..." errors.

Embedded Form:

How it works:
The embed SDK automatically detects its own script URL and uses that origin to load forms. This means:
• ✓ Works with local HTML files (file:// protocol)
• ✓ Works on any external domain
• ✓ No configuration needed—just copy and paste
• ✓ Forms always load from the correct environment
